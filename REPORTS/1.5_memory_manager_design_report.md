# 1.5 Memory Manager 设计方案

## 执行摘要

本文档详细描述了 Mini-Coding-Agent Memory Manager 的设计方案。基于代码分析，我们选择实现以下三个核心功能：

1. **Rolling Summary (滚动摘要)** - 智能压缩长对话历史
2. **Pinned Facts / Constraints (关键约束保留)** - 确保用户约束不被遗忘
3. **Tool Output Compression (工具输出压缩)** - 智能过滤噪音数据

**设计目标**: 在不显著增加延迟的前提下，将 History Token 消耗降低至 60% 以下，约束保留率提升至 90% 以上。

**设计日期**: 2025-01-27

## 1. Feature 详细设计

### 1.1 Rolling Summary (滚动摘要)

#### 1.1.1 功能描述

在长对话过程中，当 history 记录超过指定阈值时，自动调用 LLM 生成对话摘要，将多个对话回合压缩为简洁的摘要文本。

#### 1.1.2 触发条件

```python
# 配置参数
SUMMARY_TRIGGER_STEPS = 30      # 触发摘要的对话回合数
SUMMARY_MIN_STEPS = 10          # 最少累积多少回合后才允许摘要
SUMMARY_OVERLAP = 5             # 摘要时保留最近N轮不压缩
```

#### 1.1.3 摘要生成流程

```
1. 检测触发条件 (history长度 > SUMMARY_TRIGGER_STEPS)
2. 提取待摘要内容 (history[0:-SUMMARY_OVERLAP])
3. 调用 LLM 生成摘要
4. 用摘要替换原 history
5. 在 memory["summary"] 中更新最新摘要
```

#### 1.1.4 摘要模板

```
## 对话摘要 [生成时间: {timestamp}]

### 任务目标
{original_task}

### 已完成工作
{completed_steps}
  - 步骤1: 描述
  - 步骤2: 描述

### 关键决策
{key_decisions}
  - 决策1
  - 决策2

### 当前状态
{current_status}

### 遗留问题
{remaining_issues}
```

#### 1.1.5 摘要示例

```
## 对话摘要 [生成时间: 2025-01-27T10:30:00Z]

### 任务目标
实现一个二分查找算法，并编写单元测试

### 已完成工作
  - 步骤1: 创建 binary_search.py 基础框架
  - 步骤2: 实现核心查找逻辑
  - 步骤3: 运行基础测试通过

### 关键决策
  - 使用递归而非迭代实现
  - 处理空数组和目标不存在的情况

### 当前状态
核心功能已实现，测试覆盖率为 80%

### 遗留问题
需要添加边界条件测试
```

```python
class RollingSummarizer:
    def __init__(self, model_client, config):
        self.model_client = model_client
        self.trigger_steps = config.get("summary_trigger_steps", 30)
        self.min_steps = config.get("summary_min_steps", 10)
        self.overlap = config.get("summary_overlap", 5)
        self.summary_template = SUMMARY_TEMPLATE

    def should_summarize(self, history):
        """判断是否需要生成摘要"""
        return len(history) >= self.trigger_steps

    def generate_summary(self, history, original_task):
        """生成对话摘要"""
        # 1. 提取待摘要内容
        content_to_summarize = history[:-self.overlap]

        # 2. 构建摘要提示
        prompt = self.build_summary_prompt(content_to_summarize, original_task)

        # 3. 调用 LLM
        summary = self.model_client.complete(prompt, max_new_tokens=300)

        # 4. 解析和验证摘要
        return self.parse_and_validate_summary(summary)

    def build_summary_prompt(self, history, task):
        """构建摘要生成提示"""
        return f"""请为以下对话历史生成简洁摘要:

任务目标: {task}

对话历史:
{self.format_history_for_summary(history)}

请按以下格式输出:
## 对话摘要

### 任务目标
[原始任务]

### 已完成工作
[简洁描述已完成的工作]

### 关键决策
[列出关键决策点]

### 当前状态
[当前工作状态]

### 遗留问题
[如果有的话]
"""

    def compress_history(self, history, summary):
        """压缩历史记录"""
        return [{
            "role": "system",
            "content": f"[对话摘要]\n{summary}",
            "created_at": now()
        }] + history[-self.overlap:]
```

### 1.2 Pinned Facts / Constraints (关键约束保留)

#### 1.2.1 功能描述

自动识别并永久保留用户在对话中提出的约束条件（如"不要修改XX文件"、"必须使用XX库"等），确保这些约束在长对话中不会被遗忘或覆盖。

#### 1.2.2 约束关键词列表

```python
CONSTRAINT_KEYWORDS = {
    # 中文约束词
    "zh": [
        "不能", "不要", "不可", "不允许",
        "必须", "一定要", "务必要",
        "保持", "维持", "不变",
        "禁止", "忌", "不准",
        "只使用", "仅用", "只能",
        "不要改变", "不要修改", "不要删除",
        "保持不变", "维持不变",
    ],
    # 英文约束词
    "en": [
        "must not", "cannot", "should not", "do not", "don't",
        "must", "have to", "need to", "required to",
        "keep", "maintain", "preserve",
        "only use", "use only", "exclusively",
        "don't change", "don't modify", "don't delete",
        "must remain", "must keep",
    ]
}

CONSTRAINT_PATTERNS = [
    r"({kw})([^。\n]+)",  # 约束词 + 内容
    r"([^.!?]*?)({kw})([^.!?]+)",  # 内容 + 约束词 + 内容
]
```

#### 1.2.3 约束识别流程

```
1. 用户输入进入 ask()
2. 调用 ConstraintKeeper.extract_constraints(user_message)
3. 识别约束关键词
4. 提取完整约束句子
5. 存储到 pinned_constraints
6. 去重和标准化
7. 在 memory_text() 中展示固定约束
```

#### 1.2.4 实现代码结构

```python
class ConstraintKeeper:
    def __init__(self, config=None):
        self.config = config or {}
        self.keywords = CONSTRAINT_KEYWORDS
        self.max_constraints = self.config.get("max_constraints", 20)
        self.pinned_constraints = []

    def extract_constraints(self, text):
        """从文本中提取约束"""
        constraints = []

        for lang, keywords in self.keywords.items():
            for keyword in keywords:
                # 使用正则表达式匹配约束
                pattern = f".*?{keyword}[^。!?\n]*[。!?]?"
                matches = re.findall(pattern, text, re.IGNORECASE)

                for match in matches:
                    constraint = self.normalize_constraint(match.strip())
                    if constraint and self.is_valid_constraint(constraint):
                        constraints.append(constraint)

        return self.deduplicate_and_limit(constraints)

    def normalize_constraint(self, text):
        """标准化约束文本"""
        # 移除多余空白
        text = re.sub(r'\s+', ' ', text)
        # 移除引号
        text = re.sub(r'^["\'"]|["\'"]$', '', text)
        return text.strip()

    def is_valid_constraint(self, text):
        """验证是否为有效约束"""
        # 长度检查
        if len(text) < 5 or len(text) > 200:
            return False

        # 必须包含明确的约束词
        has_keyword = False
        for keywords in self.keywords.values():
            if any(kw in text.lower() for kw in keywords):
                has_keyword = True
                break

        return has_keyword

    def add_constraints(self, new_constraints):
        """添加新约束到固定列表"""
        for constraint in new_constraints:
            if constraint not in self.pinned_constraints:
                self.pinned_constraints.append(constraint)

        # 保持数量限制
        if len(self.pinned_constraints) > self.max_constraints:
            # 优先保留更早的约束
            self.pinned_constraints = self.pinned_constraints[-self.max_constraints:]

    def get_constraints_text(self):
        """获取约束文本"""
        if not self.pinned_constraints:
            return ""

        lines = ["[关键约束]", "必须遵守以下约束:"]
        for i, constraint in enumerate(self.pinned_constraints, 1):
            lines.append(f"  {i}. {constraint}")

        return "\n".join(lines)

    def serialize(self):
        """序列化约束状态"""
        return {
            "pinned_constraints": self.pinned_constraints,
            "max_constraints": self.max_constraints
        }

    @classmethod
    def deserialize(cls, data):
        """反序列化约束状态"""
        keeper = cls()
        keeper.pinned_constraints = data.get("pinned_constraints", [])
        keeper.max_constraints = data.get("max_constraints", 20)
        return keeper
```

#### 1.3.3 压缩算法

##### 策略1: 测试输出压缩

```python
def compress_test_output(self, text):
    """压缩测试输出"""
    # 统计模式
    passed_match = re.findall(r'(\d+) passed', text)
    failed_match = re.findall(r'(\d+) failed', text)
    error_match = re.findall(r'(\d+) error', text)

    passed = sum(int(m) for m in passed_match)
    failed = sum(int(m) for m in failed_match)
    errors = sum(int(m) for m in error_match)

    if passed + failed + errors > 10:
        # 大规模测试输出，提取统计信息
        summary = f"[测试统计] passed: {passed}, failed: {failed}, error: {errors}"

        # 提取关键错误信息
        errors_detail = re.findall(
            r'(FAILED|E|ERROR)[^"\']{0,200}',
            text,
            re.MULTILINE
        )

        if errors_detail:
            summary += f"\n\n[关键错误] ({len(errors_detail)} 条)\n"
            for i, error in enumerate(errors_detail[:3], 1):  # 最多显示3条
                summary += f"{i}. {error[:100]}...\n"
        elif failed == 0 and errors == 0:
            summary += "\n\n[所有测试通过]"

        return summary

    return text
```

##### 策略2: 日志压缩

```python
def compress_log_output(self, text):
    """压缩日志输出"""
    lines = text.split('\n')

    # 分类日志行
    error_lines = []
    warning_lines = []
    info_lines = []

    for line in lines:
        if 'ERROR' in line or 'FAILED' in line:
            error_lines.append(line)
        elif 'WARNING' in line or 'WARN' in line:
            warning_lines.append(line)
        elif 'INFO' in line:
            info_lines.append(line)

    # 重组输出
    result = []

    if error_lines:
        result.append(f"[ERROR] {len(error_lines)} 条错误:")
        for line in error_lines[:5]:  # 最多5条
            result.append(f"  {line[:150]}")

    if warning_lines:
        result.append(f"[WARNING] {len(warning_lines)} 条警告:")
        for line in warning_lines[:3]:  # 最多3条
            result.append(f"  {line[:150]}")

    if info_lines:
        result.append(f"[INFO] 共 {len(info_lines)} 条信息 (已省略)")

    return '\n'.join(result) if result else "[日志输出已省略]"
```

##### 策略3: 重复内容压缩

```python
def compress_repeated_lines(self, text):
    """压缩重复行"""
    lines = text.split('\n')
    compressed = []
    prev_line = None
    count = 0

    for line in lines:
        if line == prev_line:
            count += 1
        else:
            if count > 1:
                compressed.append(f"[上条内容重复 {count} 次]")
            elif prev_line is not None:
                compressed.append(prev_line)
            prev_line = line
            count = 1

    if count > 1:
        compressed.append(f"[上条内容重复 {count} 次]")
    elif prev_line is not None:
        compressed.append(prev_line)

    return '\n'.join(compressed)
```

##### 策略4: 智能截断

```python
def smart_truncate(self, text, max_length):
    """智能截断，保留关键信息"""
    if len(text) <= max_length:
        return text

    # 尝试保留开头和结尾
    head_length = int(max_length * 0.7)
    tail_length = max_length - head_length - 10

    head = text[:head_length]
    tail = text[-tail_length:] if tail_length > 0 else ""

    # 查找中间是否有重要信息
    middle = text[head_length:-tail_length] if tail_length > 0 else text[head_length:]

    # 如果中间有关键词，尝试保留
    important_keywords = ['error', 'failed', 'exception', 'traceback']
    important_parts = []

    for keyword in important_keywords:
        matches = re.finditer(keyword, middle, re.IGNORECASE)
        for match in matches:
            start = max(0, match.start() - 50)
            end = min(len(middle), match.end() + 50)
            important_parts.append(middle[start:end])

    result = head
    if important_parts:
        result += "\n...[包含关键信息]...\n"
        result += "\n".join(important_parts[:2])  # 最多2段
    if tail:
        result += "\n..." + tail

    return result
```

#### 1.3.4 压缩执行流程

```python
class ToolOutputCompressor:
    def compress(self, tool_name, output):
        """压缩工具输出"""
        original_length = len(output)

        # 1. 根据工具类型选择压缩策略
        if tool_name == "run_shell":
            compressed = self.compress_shell_output(output)
        elif tool_name == "search":
            compressed = self.compress_search_output(output)
        elif tool_name in ["read_file", "list_files"]:
            compressed = self.compress_file_output(output)
        else:
            compressed = output

        # 2. 应用最大长度限制
        compressed = self.smart_truncate(compressed, self.max_output_length)

        # 3. 更新统计
        self.update_stats(original_length, len(compressed))

        return compressed

    def compress_shell_output(self, output):
        """压缩 shell 命令输出"""
        # 检测是否为测试输出
        if 'test' in output.lower() and ('passed' in output or 'failed' in output):
            return self.compress_test_output(output)

        # 检测是否为日志输出
        if any(marker in output for marker in ['[INFO]', '[DEBUG]', '[ERROR]', 'ERROR:']):
            return self.compress_log_output(output)

        # 默认压缩重复内容
        return self.compress_repeated_lines(output)

    def update_stats(self, original, compressed):
        """更新压缩统计"""
        self.compression_stats["total_compressions"] += 1
        self.compression_stats["total_original_length"] += original
        self.compression_stats["total_compressed_length"] += compressed

    def get_compression_ratio(self):
        """获取压缩比"""
        if self.compression_stats["total_original_length"] == 0:
            return 1.0
        return (
            self.compression_stats["total_compressed_length"] /
            self.compression_stats["total_original_length"]
        )
```

## 2. Memory 分层结构设计

### 2.1 层次架构

```
Memory
├── Layer 1: Pinned Layer (永久层)
│   ├── constraints (关键约束)
│   └── original_task (原始任务)
│
├── Layer 2: Summary Layer (摘要层)
│   ├── current_summary (当前摘要)
│   └── previous_summaries (历史摘要列表)
│
├── Layer 3: Working Layer (工作层)
│   ├── files (最近文件)
│   ├── notes (笔记)
│   └── recent_history (未压缩的历史)
│
└── Layer 4: Raw Layer (原始层)
    └── full_history (完整历史，可选)
```

### 2.2 数据结构定义

```python
class MemoryLayer:
    """Memory 分层结构"""

    # Layer 1: Pinned Layer - 永久保留
    PINNED_LAYER_KEYS = ["constraints", "original_task"]

    # Layer 2: Summary Layer - 摘要信息
    SUMMARY_LAYER_KEYS = ["current_summary", "summary_history"]

    # Layer 3: Working Layer - 当前工作状态
    WORKING_LAYER_KEYS = ["files", "notes", "recent_history"]

    # Layer 4: Raw Layer - 完整历史（可选）
    RAW_LAYER_KEYS = ["full_history"]

    def __init__(self):
        self.pinned = {
            "constraints": [],
            "original_task": ""
        }

        self.summary = {
            "current_summary": "",
            "summary_history": [],
            "last_summary_step": 0
        }

        self.working = {
            "files": [],
            "notes": [],
            "recent_history": []
        }

        self.raw = {
            "full_history": []
        }

        # 配置
        self.config = {
            "max_constraints": 20,
            "max_files": 8,
            "max_notes": 5,
            "max_recent_history": 20,
            "summary_trigger_steps": 30,
            "summary_overlap": 5
        }

    def to_session_dict(self):
        """转换为 session 存储格式"""
        return {
            "task": self.pinned["original_task"],
            "summary": self.summary["current_summary"],
            "constraints": self.pinned["constraints"],
            "files": self.working["files"],
            "notes": self.working["notes"],
            "compression_stats": {
                "summary_trigger_steps": self.summary["last_summary_step"]
            }
        }

    @classmethod
    def from_session_dict(cls, session_dict):
        """从 session 恢复"""
        layer = cls()
        layer.pinned["original_task"] = session_dict.get("task", "")
        layer.pinned["constraints"] = session_dict.get("constraints", [])
        layer.summary["current_summary"] = session_dict.get("summary", "")
        layer.working["files"] = session_dict.get("files", [])
        layer.working["notes"] = session_dict.get("notes", [])
        return layer
```

```python
class MemoryManager:
    """
    Memory Manager - 统一管理所有 Memory 优化功能

    功能:

        self.tool_compressor = ToolOutputCompressor(self.config)

        # Memory 状态
        self.memory_layer = MemoryLayer()
        self.history = []

        # 统计
        self.stats = {
            "total_constraints_found": 0,
            "total_summaries_generated": 0,
            "total_tools_compressed": 0,
            "compression_ratio": 1.0
        }

    # ==================== 核心接口 ====================

    def process_user_input(self, user_message):
        """
        处理用户输入，提取约束等信息

        Args:
            user_message: 用户输入文本

        Returns:
            处理后的用户输入（可能添加标记）
        """
        # 1. 提取约束
        constraints = self.constraint_keeper.extract_constraints(user_message)
        if constraints:
            self.constraint_keeper.add_constraints(constraints)
            self.stats["total_constraints_found"] += len(constraints)

        # 2. 更新任务（如果是新任务）
        if not self.memory_layer.pinned["original_task"]:
            self.memory_layer.pinned["original_task"] = user_message[:300]

        return user_message

    def compress_tool_output(self, tool_name, output):
        """
        压缩工具输出

        Args:
            tool_name: 工具名称
            output: 原始输出

        Returns:
            压缩后的输出
        """
        compressed = self.tool_compressor.compress(tool_name, output)

        if compressed != output:
            self.stats["total_tools_compressed"] += 1

        return compressed

    def check_and_generate_summary(self, history):
        """
        检查是否需要生成摘要

        Args:
            history: 对话历史

        Returns:
            (是否生成了摘要, 摘要内容)
        """
        if self.rolling_summarizer.should_summarize(history):
            summary = self.rolling_summarizer.generate_summary(
                history,
                self.memory_layer.pinned["original_task"]
            )

            if summary:
                # 更新 memory layer
                self.memory_layer.summary["current_summary"] = summary
                self.memory_layer.summary["last_summary_step"] = len(history)

                # 压缩 history
                compressed_history = self.rolling_summarizer.compress_history(
                    history,
                    summary
                )

                self.stats["total_summaries_generated"] += 1

                return True, summary

        return False, ""

    def get_memory_text(self):
        """
        获取当前 Memory 文本

        Returns:
            格式化后的 memory 文本
        """
        lines = ["Memory:"]

        # 1. Task
        task = self.memory_layer.pinned["original_task"]
        lines.append(f"- task: {task or '-'}")

        # 2. Constraints (Pinned Layer)
        constraints_text = self.constraint_keeper.get_constraints_text()
        if constraints_text:
            lines.append("")
            lines.append(constraints_text)

        # 3. Summary (Summary Layer)
        summary = self.memory_layer.summary["current_summary"]
        if summary:
            lines.append("")
            lines.append(f"[对话摘要]\n{summary}")

        # 4. Files (Working Layer)
        files = ", ".join(self.memory_layer.working["files"]) or "-"
        lines.append(f"- files: {files}")

        # 5. Notes (Working Layer)
        notes = "\n".join(
            f"- {note}" for note in self.memory_layer.working["notes"]
        ) or "-"
        lines.append("- notes:")
        lines.append(notes)

        return "\n".join(lines)

    def update_stats(self):
        """更新统计信息"""
        self.stats["compression_ratio"] = self.tool_compressor.get_compression_ratio()

    def get_stats(self):
        """获取统计信息"""
        self.update_stats()
        return {
            **self.stats,
            "compression_ratio": round(self.stats["compression_ratio"], 3)
        }

    def serialize(self):
        """序列化 Memory 状态"""
        return {
            "memory_layer": {
                "pinned": self.memory_layer.pinned,
                "summary": self.memory_layer.summary,
                "working": self.memory_layer.working
            },
            "stats": self.stats,
            "compressor_stats": self.tool_compressor.compression_stats
        }

    @classmethod
    def deserialize(cls, data, model_client):
        """反序列化 Memory 状态"""
        manager = cls(model_client)

        # 恢复 memory layer
        if "memory_layer" in data:
            ml_data = data["memory_layer"]
            manager.memory_layer.pinned = ml_data.get("pinned", manager.memory_layer.pinned)
            manager.memory_layer.summary = ml_data.get("summary", manager.memory_layer.summary)
            manager.memory_layer.working = ml_data.get("working", manager.memory_layer.working)

        # 恢复 constraint keeper
        if "memory_layer" in data and "pinned" in data["memory_layer"]:
            constraints = data["memory_layer"]["pinned"].get("constraints", [])
            manager.constraint_keeper.pinned_constraints = constraints

        # 恢复统计
        if "stats" in data:
            manager.stats = data["stats"]

        if "compressor_stats" in data:
            manager.tool_compressor.compression_stats = data["compressor_stats"]

        return manager

    def _normalize_config(self, config):
        """标准化配置"""
        default_config = {
            "summary_trigger_steps": 30,
            "summary_min_steps": 10,
            "summary_overlap": 5,
            "max_constraints": 20,
            "max_files": 8,
            "max_notes": 5,
            "max_output_length": 500,
            "test_output_max": 50,
            "error_output_max": 300
        }

        if config:
            default_config.update(config)

        return default_config
```

## 6. 向后兼容性设计

### 6.1 Session 迁移

```python
def migrate_session(old_session):
    """
    迁移旧版本的 session 到新版本

    旧版本 memory 结构:
    {
        "task": "",
        "files": [],
        "notes": []
    }

    新版本 memory 结构:
    {
        "task": "",
        "summary": "",
        "constraints": [],
        "files": [],
        "notes": [],
        "compression_stats": {}
    }
    """
    new_session = copy.deepcopy(old_session)

    # 1. Memory 迁移
    if "memory" in new_session:
        old_memory = new_session["memory"]

        # 添加新字段
        new_session["memory"]["summary"] = old_memory.get("summary", "")
        new_session["memory"]["constraints"] = old_memory.get("constraints", [])
        new_session["memory"]["compression_stats"] = {
            "summary_trigger_steps": 0,
            "last_summary_step": 0
        }

    return new_session
```
