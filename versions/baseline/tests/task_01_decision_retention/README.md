# Task 1: Decision Retention 测试

## 任务描述

**目标**: 测试关键约束和决策保留能力

**考察点**:
1. 关键约束保留能力
2. 决策与普通历史的分离能力
3. 最终总结的准确性
4. 实现与约束一致性

## 测试文件

- `test_task1.py`: 自动化测试脚本
- `README.md`: 本文件

## 运行方法

```bash
# 激活 conda 环境
conda activate minicodeagent

# 运行测试
cd versions/baseline/tests/task_01_decision_retention
python test_task1.py
```

## 测试场景

1. 在早期轮次提出关键约束（如兼容性要求）
2. 执行多轮对话
3. 在后期轮次检查约束是否被保留
4. 验证最终总结是否包含约束

## 预期结果

### Baseline 版本（无优化）

**预期行为**:
- 关键约束可能在后期轮次中被遗忘
- 最终总结可能不包含约束信息
- 上下文长度随对话增加

**预期指标**:
- 关键约束保留率: 低（< 50%）
- 上下文长度: 快速增长

### With-Memory 版本（优化后）

**预期行为**:
- 关键约束被显式提取并保护
- 最终总结准确包含约束信息
- 上下文长度得到控制

**预期指标**:
- 关键约束保留率: 高（> 90%）
- 上下文长度: 得到控制
