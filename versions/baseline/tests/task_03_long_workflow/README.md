# Task 3: Long Workflow 测试

## 任务描述

**目标**: 测试任务状态分段管理能力

**考察点**:
1. 是否能保留任务阶段状态
2. 是否能保留"当前方案已经做到哪一步"
3. 是否能最终总结出正确的文件变更与剩余风险

## 测试文件

- `test_task3.py`: 自动化测试脚本
- `README.md`: 本文件

## 运行方法

```bash
# 激活 conda 环境
conda activate minicodeagent

# 运行测试
cd versions/baseline/tests/task_03_long_workflow
python test_task3.py
```

## 测试场景

1. 执行多阶段 workflow（探索 → 实现 → 测试 → 修复）
2. 在中途检查 agent 是否知道当前状态
3. 在最后验证总结是否准确

## 预期结果

### Baseline 版本（无优化）

**预期行为**:
- Agent 可能在中途忘记已完成的工作
- 当前阶段状态不清晰
- 最终总结可能不准确

**预期指标**:
- 阶段识别准确率: 低
- 最终总结准确率: 低

### With-Memory 版本（优化后）

**预期行为**:
- 显式记录任务阶段状态
- 保留已完成工作的进度
- 最终总结准确完整

**预期指标**:
- 阶段识别准确率: 高
- 最终总结准确率: 高
