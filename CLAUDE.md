# Trae 用户习惯记录

## 工作流程习惯

### 每次对话后需要做的事情

1. **更新 todo list**
   - 使用 TodoWrite 工具更新任务进度
   - 标记已完成的任务
   - 记录下一步任务

2. **更新对话记录**
   - 记录重要的用户指令
   - 记录项目关键决策
   - 记录遇到的问题和解决方案

## 项目相关习惯

### Memory-Aware Coding Agent 项目

- 使用 todo_list.md 跟踪项目进度
- 按照 Phase 顺序执行：
  - Phase 1: 项目准备
  - Phase 2: 设计
  - Phase 3: 实现
  - Phase 4: 文档
  - Phase 5: 测试
  - Phase 6: 结果报告

## 技术偏好

- 使用中文进行交流
- 偏好结构化的任务分解
- 重视测试和验证
- 关注代码质量和可读性

## 常用工具

- TodoWrite - 任务管理
- Read/Write - 文件操作
- RunCommand - 命令执行
- SearchCodebase - 代码搜索
